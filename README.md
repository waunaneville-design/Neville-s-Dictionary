# Wordly Dictionary SPA

A lightweight single-page dictionary application built with HTML, CSS, and JavaScript.

## Overview

This app lets users search for words, view definitions, pronunciation, synonyms, and source information without reloading the page. Favorites are saved locally so users can revisit saved terms instantly.

## Features

- Search for words using a live JavaScript form
- Fetch definitions from the free Dictionary API
- Display pronunciation, part of speech, examples, and synonyms
- Save favorite words to `localStorage`
- Toggle between light and dark themes
- Error messaging for missing or invalid search results

## Files

- `index.html` — app template and UI structure
- `styles.css` — responsive styling and theme support
- `script.js` — SPA behavior, API calls, favorites handling
- `script.json` — project metadata

## Usage

1. Open `index.html` in your browser.
2. Enter a word in the search field.
3. Click `Search` to view the definition and details.
4. Click `Save word` to add the term to favorites.

## Notes

- Favorites and theme selection persist across page reloads.
- The app uses the public Dictionary API at `https://api.dictionaryapi.dev/`.
